# udacity_fullstack_project3 (Catalogapp).

##Instructions ##
* Install Vagrant and VirtualBox.
* Clone the fullstack-nanodegree-vm repository
* Launch the Vagrant VM
* Copy all the files provided and past it to the catalog directory.
* Change data-clientid with yours' in login.html, also download client_secrets.json file.
* Create the  data base using "python datbase_setup.py".
* Add data to the database using "python add_data.py".
* Run project.py file using "python project.py" to get server running.
* Go "localhost:8000" using your browser and play around with the app.
* You may need to include:
* [flask](http://flask.pocoo.org) 
* [sqlalchemy](http://www.sqlalchemy.org)
